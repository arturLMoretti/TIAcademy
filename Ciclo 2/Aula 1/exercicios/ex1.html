Ciclo 2 - Exercício 1

1 - Esta afirmação está correta: "O JS é uma linguagem interpretada, portanto irá gerar um código fonte. Isto está correto". Explique.

R: O JavaScript é uma linguagem interpretada, então necessita de um motor para executar os comandos comandos inseridos, não gerando um código fonte.

2 - Ainda nos dias de hoje o JS só pode ser usado por meio de um browser.

R: Apesar de todo navegador possuir motor para interpretar o JavaScript, é possível usar o JS por meio de um framework, que não necessita de navegador.
